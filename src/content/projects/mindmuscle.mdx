## Description

A React Native coaching platform built on a domain-plugin architecture where the entire content taxonomy — terminology, categories, drills, metadata schemas — is swappable via a single config file. Ships with 13 sport psychology drills and 3 periodized programs, backed by Firebase Auth, Cloud Firestore, and Zustand with full offline support. Adding a new coaching domain (e.g. nutrition, academic tutoring) requires zero changes to core infrastructure.
