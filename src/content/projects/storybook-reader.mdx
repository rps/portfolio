## Description

A React Native iPad app purpose-built for children with Cerebral Visual Impairment (CVI), enforcing a fixed 55/45 landscape image-text split for predictable, distraction-free reading. A custom Expo config plugin locks iOS orientation, and a discriminated union page type lets single-book and series viewers share identical FlatList rendering. Connects to the Storybook Library API on Fly.io with content served from Supabase and Cloudflare R2.
