## Description

A full-stack AI pipeline that transforms public domain stories into photorealistically illustrated children's books for readers with Cerebral Visual Impairment. A Hono/TypeScript API on Fly.io orchestrates OpenAI GPT-4.1-mini for text segmentation and GPT-image-1-mini for image generation, with Upstash Redis job queuing, Supabase persistence, and Cloudflare R2 storage. The deployed API is read-only — the AI worker runs locally for cost control.
